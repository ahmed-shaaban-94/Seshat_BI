# Keyboard and focus review checklist

Review each report page against these items and record the result in the
design-review evidence. Guidance only -- it never grants a readiness pass.

- Every interactive element (slicer, tab, drill control) is reachable by Tab.
- Focus order follows the visual reading order.
- Focus is visible: a keyboard user always knows where they are.
- Tooltips are not the only place a value or definition appears.
